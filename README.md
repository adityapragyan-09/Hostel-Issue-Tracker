# 🏢 Hostel Complaint Management System

A real-time hostel complaint management platform built for students and hostel administration.

The system includes:

- 👨‍🎓 Student Portal
- 👨‍💼 Admin Dashboard
- ⚡ Real-time complaint tracking
- 📷 Media uploads
- 🔐 Secure Supabase authentication

## ✨ Features

### 👨‍🎓 Student Panel
- NIAT ID based identification
- Auto-filled student names
- Complaint submission system
- Image/video uploads
- Real-time complaint updates
- Complaint history tracking
- Dark mode support

### 👨‍💼 Admin Panel
- Secure admin authentication
- View all complaints
- Status management
- Admin remarks system
- Real-time updates
- Soft delete support
- Export complaint data
